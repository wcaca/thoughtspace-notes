# docs/superpowers/specs/
> L3 目录说明 - 父级: ../CLAUDE.md

## 用途
存放已批准的设计文档(spec)。每个 spec 描述"为什么这么做、做什么、不做什么"。

## 命名约定
- `YYYY-MM-DD-<topic>-design.md` — 例:`2026-07-05-phase-0-lightweight-restructure-design.md`

## 与 plans/ 的关系
- spec 批准后,由 writing-plans skill 生成对应 plan,plan 路径:`docs/superpowers/plans/YYYY-MM-DD-<feature>.md`
